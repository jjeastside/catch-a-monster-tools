Hidden Treasure Pattern Finder

Click squares manually or generate patterns that hit hidden rectangular treasures.

Hover a square to see coverage gain.
How it works

A click succeeds if it lands anywhere inside the hidden treasure rectangle.

The app checks every possible treasure placement. Green overlay means that placement is hit. Red overlay means missed.

Exact optimal searches for the fewest clicks that reach the target coverage. The Step button follows the selected mode, so Exact optimal steps through an optimal solution.
