# Catch a Monster Tools

Community-made calculators, solvers, and optimization tools for Catch a Monster.

## Current Tools

### Hidden Treasure Solver

Find optimal search patterns for hidden treasure events.

Features:

- Exact Optimal Solver
- Greedy Solver
- Pretty Guarantee Patterns
- Step-by-Step Solving
- Placement Coverage Visualization
- Coverage Statistics
- Manual Pattern Editing
- Pattern Exporting

## Planned Tools
