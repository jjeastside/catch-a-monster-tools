# Catch a Monster Tools

A static website for Catch a Monster calculators and solvers.

## Pages

- Home: `index.html`
- Hidden Treasure Solver: `tools/treasure-solver/index.html`

## Publish on GitHub Pages

1. Create a GitHub repository named `catch-a-monster-tools`.
2. Upload everything inside this folder.
3. Go to `Settings → Pages`.
4. Set:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
5. Save.

Your site URL will be:

`https://YOUR_USERNAME.github.io/catch-a-monster-tools/`

The solver page will be:

`https://YOUR_USERNAME.github.io/catch-a-monster-tools/tools/treasure-solver/`
